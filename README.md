# County-Equitable-Share-Analysis in Kenya.
## Project overview
This project analyses the distribution of kenyas county equitable share allocation across all 47 counties using python and jupyter notebook.
## Objectives
-Analyse county equitable share allocations.
-Compare funding across counties.
-Identify counties receiving the highest and lowest allocations.
-Visualize allocation patterns.
-Generate data driven insights.
## Tools used
-Python.
-Jupyter Notebook.
-Pandas.
-NumPy.
-Matplotlib.
-Seaborn.
-Sckitlearn.
